# 環境センサーボード(BMP280/BME280)

[BMP280/BME280](https://www.bosch-sensortec.com/products/environmental-sensors/humidity-sensors-bme280/)は温度、湿度、気圧などのデータが取得可能なセンサーです。このうち、BMP280は温度と湿度のみ、BME280はBMP280に追加して気圧が取得可能です。
